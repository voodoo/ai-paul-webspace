# PRD: Paul's Personal Site v1.0

## Overview
Minimal personal landing page with a single call-to-action.

## Goal
Get visitors to book a call with Paul.

## Requirements

### Hero Section
- Full viewport height
- Paul's name centered (large, bold)
- Clean, modern typography

### Call-to-Action
- Single button below name
- Links to: https://cal.com/paul-vudmaska/15min
- Button text: "Book a Call"

### Design
- Minimal — no nav, no footer, no clutter
- Dark/light theme toggle (top right corner)
- Default: system preference
- Mobile responsive

### Colors
- **Dark theme:** Dark blue background, gold accents/trim
- **Light theme:** Off-white background, dark gray text

### Tech Stack
- Astro
- Static output
- Fast, lightweight

## Out of Scope (v1)
- About section
- Portfolio
- Blog
- Contact form

## Deliverables
- `index.html`
- `styles.css`
- Deployed to GitHub Pages (optional)
